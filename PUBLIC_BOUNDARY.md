# Public Boundary

This repository is a public education artifact.

It contains a small, clean-room active inference primer using synthetic arrays and toy/spec examples only.

## Included

- Minimal free-energy utility functions
- Small synthetic numerical tests
- Educational notes and limitations
- Public references

## Excluded

This repository does not disclose:

- private AI systems
- private implementation details
- production logs
- deployment receipts
- proprietary architecture
- internal governance mechanisms
- customer, user, or partner data
- credentials, keys, or operational secrets

## Safe Use

The material here can be cited as educational code and public research scaffolding. It should not be cited as evidence that any deployed system is safe, aligned, conscious, production-ready, or formally validated.

Any practical active inference system would require separate mathematical review, empirical validation, security review, and deployment-specific testing.
